# gibasa: An Alternative 'Rcpp' Wrapper of 'MeCab'

A plain 'Rcpp' wrapper for 'MeCab' that can segment Chinese, Japanese,
and Korean text into tokens. The main goal of this package is to provide
an alternative to 'tidytext' using morphological analysis.
